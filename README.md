  <h1>ChatGPT를 이용한 반려동물 정보 추천 서비스</h1>
  <h2>1. 목표와 기능</h2>
   
  <h3>1-1 목표</h3>
 <ul>
  <li> 2023년 기준 개, 고양이만해도 약 800만 마리 이상의 반려동물과 함께 살아가고있습니다.
    동물을 좋아하지만 키우지 못하시는분, 초보 반려인들에게 유용한 정보를 제공하는 것이 주요 목표입니다. </li>
 </ul>

  <h3>1-2 기능</h3>
  <ul>
    <li>ChatGPT API를 통하여 원하시는 정보를 검색하실 수 있습니다.</li>
    <li>검색하신 정보를 특징, 장/단점, 주의사항, 주변시설 등으로 보여드립니다.</li>
  </ul>
